# Assets

Copy assets from the original poker-design project:

1. Copy `poker-design/src/assets/logo.png` → `public/assets/logo.png`
2. Copy all files from `poker-design/src/assets/images/` → `public/assets/images/`
3. Copy all files from `poker-design/src/assets/videos/` → `public/assets/videos/`

Required for the site: logo.png, images (footer.webp, imgi_*.jpg), and videos (top-video.mov, medium-video.mov).
